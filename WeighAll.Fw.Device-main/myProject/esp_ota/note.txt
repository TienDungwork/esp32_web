Tóm tắt luồng:
- App chính nhận API_UPDATE_FIRMWARE → ghi URL + cờ + set boot agent → restart.
- Bootloader → boot vào ota_agent (factory).
- ota_agent đọc URL → tải firmware → flash vào ota_0/ota_1 → xóa cờ → set boot partition → restart.
- Bootloader boot app mới.
Tạo 2 project (app chính và agent), flash cả hai lần đầu. Sau đó mọi OTA app chính sẽ qua agent.
